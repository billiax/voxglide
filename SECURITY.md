# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do NOT open public issues for security vulnerabilities.**

Use GitHub's [private vulnerability reporting](https://github.com/billiax/voxglide/security/advisories/new) feature to report security issues.

### Response Timeline

- **Acknowledge:** Within 48 hours
- **Status update:** Within 7 days
- **Fix for critical issues:** Within 30 days

### Scope

In scope:
- SDK client-side code
- Server proxy
- WebSocket protocol
- API key handling and session management

Out of scope:
- Browser Speech API bugs
- Third-party LLM provider vulnerabilities
- Issues in example/demo pages

## Disclosure

We follow coordinated disclosure. We will credit reporters in the release notes (unless anonymity is requested).
